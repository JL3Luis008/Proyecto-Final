# Proyecto-Final

## 🎯 Objetivo

Crear un ecommerce desde cero para adquirir el conocimiento  y practicar de los diferentes componentes con los que esta desarrollado. este repositorio esta dividido en dos partes (Ecommerce-Api y Ecommerce-App) 

---
## Datos de proyecto

Nombre de ecommerce: Retro-Bits.com

Giro: Compra y venta de consolas y videojuegos retro y clasicos.

publico objetivo: Se busca el interes de amantes de los video juegos que busquen titulos que marcaron su infancia o que busquen los origenes de sus franquicias favoritas. entregando productos de excelente calidad, testeados y acondicionados para que perduren por generaciones.

## 🚀 Cómo Ejecutar

### Prerequisitos
- **Node.js** (v20 o superior)
- **npm** (incluido con Node.js)

### Instalación y Ejecución

```bash
# Navegar al proyecto
cd Proyecto-Final

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm start
```

### Scripts Disponibles

```bash
npm start          # Servidor de desarrollo (http://localhost:3000)
npm run build      # Construcción para producción
npm test           # Ejecutar tests
npm run eject      # Exponer configuración (no recomendado)
```
