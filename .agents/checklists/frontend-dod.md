# Definition of Done: Frontend

- [ ] UI es responsiva (Desktop / Mobile).
- [ ] Contraste de colores verificado (a11y).
- [ ] No hay errores de consola.
- [ ] Hooks personalizados extraídos si hay lógica compleja.
- [ ] Prop-types o validaciones de tipo presentes.
- [ ] Asset optimization (imágenes comprimidas, iconos SVG).
- [ ] Loading states implementados.
- [ ] Error boundary o manejo de fallos UI.
