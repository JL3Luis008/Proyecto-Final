# Definition of Done: Backend

- [ ] Lógica centralizada en Servicios (no en controladores).
- [ ] Manejo de errores con `catchAsync`.
- [ ] Validación de Schemas (Joi/Zod o Mongoose).
- [ ] Logs informativos implementados.
- [ ] Sin secretos en el código.
- [ ] Test de integración para el nuevo endpoint.
- [ ] Documentación de API actualizada.
- [ ] Índices creados si el performance lo requiere.
